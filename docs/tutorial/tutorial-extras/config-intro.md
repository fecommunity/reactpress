title: 项目配置项
---

项目启动后会默认加载 `.env` 文件，主要支持的配置项如下：

```bash
# 数据库主机
DB_HOST=127.0.0.1 

# 数据库端口
DB_PORT=3306

# 数据库用户名
DB_USER=reactpress

# 数据库密码
DB_PASSWD=reactpress

# 数据库名称
DB_DATABASE=reactpress

# 客户端地址
CLIENT_SITE_URL=http://localhost:3001

# 服务端地址
SERVER_SITE_URL=http://localhost:3002

# Github OAuth 登录 Id
GITHUB_CLIENT_ID=0 

 # Github OAuth 登录 Secret
GITHUB_CLIENT_SECRET=0
```
