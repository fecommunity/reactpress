# 联系我们

欢迎就 ReactPress 提出问题、报告 Bug、分享想法或参与贡献。

## 支持渠道

| 渠道 | 适用场景 | 链接 |
| --- | --- | --- |
| **GitHub Issues** | Bug、可复现问题 | [github.com/fecommunity/reactpress/issues](https://github.com/fecommunity/reactpress/issues) |
| **GitHub Discussions** | 提问、想法、展示 | [github.com/fecommunity/reactpress/discussions](https://github.com/fecommunity/reactpress/discussions) |
| **邮件** | 私人咨询、安全报告 | [admin@gaoredu.com](mailto:admin@gaoredu.com) |

## 提交 Issue 前

1. 在您的环境运行 `reactpress doctor` 与 `reactpress status`
2. 搜索[已有 Issue](https://github.com/fecommunity/reactpress/issues) 避免重复
3. 阅读 [FAQ](/zh/docs/reference/faq) 与[故障排查](/zh/docs/reference/troubleshooting)
4. 附上 Node 版本、操作系统、ReactPress 版本与相关日志

可参考[《提问的智慧》](https://github.com/ryanhanwu/How-To-Ask-Questions-The-Smart-Way)提高回复效率。

## 安全报告

如发现安全漏洞，请**不要**公开 Issue，发邮件至 [admin@gaoredu.com](mailto:admin@gaoredu.com)，并在公开披露前留出合理修复时间。

## 参与贡献

欢迎向 [GitHub](https://github.com/fecommunity/reactpress/pulls) 提交 Pull Request。文档可直接通过任意页面的 **Edit this page** 编辑。
