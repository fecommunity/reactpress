## ⚡ Why ReactPress

Modern content systems are broken:

* WordPress → powerful but heavy, plugin-driven, slow to evolve
* Headless CMS → flexible but fragmented and expensive to assemble
* Static site generators → fast, but no real CMS for editors

---

### ReactPress fixes this.

> **Everything you need to build a modern publishing system — in one CLI.**

---

## ⚡ 60 seconds to your first CMS

```bash
npm i -g @fecommunity/reactpress
mkdir my-blog && cd my-blog

reactpress init
reactpress dev
```

Then you instantly get:

* 🧠 Production-ready CMS backend (API)
* 📝 Admin dashboard for content editing
* 🗂 Posts / pages / tags / categories / media
* ⚡ RESTful API out of the box
* 🐳 Docker-ready database (optional)
* 🚀 Deployment-ready architecture

## 🧱 Architecture

| Layer            | Role                             |
| ---------------- | -------------------------------- |
| CMS Backend      | Content storage + API            |
| Admin Panel      | Writing & management UI          |
| CLI              | Setup, dev, build, deploy        |
| Frontend (Theme) | Public website (Next.js / React) |
| Theme Starter    | Official production UI           |

> Fully **headless**, fully **decoupled**, fully **customizable**.

## 🚀 Quick Start

```bash
npm i -g @fecommunity/reactpress
mkdir my-blog && cd my-blog

reactpress init
reactpress dev
```

Open:

* API: [http://localhost:3002/api](http://localhost:3002/api)
* Admin: [http://localhost:3002](http://localhost:3002)
