# Repository root static assets

Marketing and demo assets live at the repository root (e.g. `usage.gif`, `poster.png`).

Brand and site icons are generated by `pnpm export:brand` and organized by type:

| Directory | Contents |
| :--- | :--- |
| `brand/` | Logo and wordmark sources: `logo.svg`, `wordmark.svg`, `logo.png`, etc. |
| `favicon/` | Browser favicons: `favicon.ico`, `favicon-16/32/48.png` |
| `icons/` | PWA / Apple Touch: `apple-touch-icon.png`, `icon-192/512.png` |

Runtime apps (`web/public`, `server/public`, etc.) still use flat paths for deployment convenience.
